## 📋 Descrição

Interface frontend moderna e responsiva para o sistema **WEG Granja**, desenvolvida com as tecnologias mais recentes do ecossistema React. Este projeto oferece uma experiência de usuário intuitiva e performática para gerenciamento de operações em ambiente de granja.

---

## 🚀 Tecnologias

- **[Next.js](https://nextjs.org/)** 16.2.6 - Framework React com SSR e SSG
- **[React](https://reactjs.org/)** 19.2.4 - Biblioteca UI
- **[Tailwind CSS](https://tailwindcss.com/)** 4 - Framework CSS utility-first
- **[PostCSS](https://postcss.org/)** - Processamento de CSS
- **[ESLint](https://eslint.org/)** 9 - Linter JavaScript

## 📦 Instalação

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Passo a passo

1. **Clone o repositório**
```bash
git clone https://github.com/viviihyy/WEG-Granja-Front.git
cd WEG-Granja-Front
```

2. **Instale as dependências**
```bash
npm install
# ou
yarn install
```

3. **Configure as variáveis de ambiente** (se necessário)
```bash
cp .env.example .env.local
```

---

## 🏃 Como Executar

### Desenvolvimento
```bash
npm run dev
```
A aplicação estará disponível em `http://localhost:3000`

### Build para Produção
```bash
npm run build
npm start
```

### Lint
```bash
npm run lint
```
